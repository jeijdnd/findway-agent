# FindWay Agent

建筑导视标识设计AI助手

## 开发角色

- **PM（WorkBuddy）**：管控方向、审查进度
- **Dev（Continue）**：按 PROGRESS_BOARD.md 写代码

## 文档

| 文件 | 说明 |
|------|------|
| `PRD.md` | 产品需求文档 |
| `DEVELOPMENT_ROADMAP.md` | 开发路线图 |
| `PROGRESS_BOARD.md` | 进度看板 |
| `SESSION_REPORT.md` | 会话报告 |
| `.continuerules` | Continue自动规则 |
| `CONTINUE_START.md` | 复制粘贴给Continue的启动语 |
