### 实现

#### 右侧按钮条（RightSidebar.jsx）
- 固定 45px 宽，竖向排列按钮
- 按钮：清单同步 / 新建项目 / 项目管理 / 设置
- 点击按钮 → 右侧滑出对应面板

#### 右侧面板（RightPanel.jsx）
- 从右边缘向左滑入（CSS transition, transform: translateX）
- 占屏幕宽度 33%
- 面板顶部有关闭按钮
- 点击面板外区域也能关闭
- 不同按钮显示不同内容

#### 修改 WorkBuddyLayout.jsx
- 现有布局改为：左侧(250px) | 主内容(flex) | 右侧按钮条(45px) | 可选面板(33% 滑入)
- 右侧面板用绝对定位 + z-index

#### 面板内容
- 清单同步 → 同步/对比清单界面（已有 compare_list 逻辑）
- 新建项目 → 新建项目表单（已有 Dashboard 里的表单）
- 项目管理 → 项目列表快速跳转
- 设置 → 打开设置页
